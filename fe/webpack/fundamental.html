# 安装

TIP

webpack4 需要安装webpack和webpack-cli两个包

npm i webpack webpack-cli -D

# 配置

默认配置文件名为根目录下的webpack.config.js，默认导出一个webpack配置对象。

# webpack配置对象

  • entry：入口文件路径
  • output：出口。
    • filename：出口文件名
    • path：必须是绝对路径
  • mode：模式。默认为production，可选development
