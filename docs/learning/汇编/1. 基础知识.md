## 汇编语言的组成
- 汇编指令：机器码的助记符，有对应的机器码
- 编译器识别符号
  - 伪指令：编译器执行
  - 其他符号：编译器识别
## CPU 对存储器的读写
CPU 要对数据进行读写，需要和外部器件进行以下三类信息的交换
- 地址信息：存储单元的地址
- 数据信息
- 控制信息：读写命令
